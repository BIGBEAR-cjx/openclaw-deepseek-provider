## 2. V2EX 推广帖

### 标题
发布了一个开源项目：DeepSeek Official Provider for OpenClaw，帮你节省 30% API 成本

### 正文
```
大家好，

最近做了一个小项目，觉得可能对这里有远程工作需求的朋友有帮助。

## 项目介绍

DeepSeek Official Provider 是一个 OpenClaw 的 Provider 插件，让你可以直接使用 DeepSeek 官方 API，而不是通过 HuggingFace 路由。

## 为什么做这个？

现在 OpenClaw 用户要使用 DeepSeek 模型，大多是通过 HuggingFace Inference：
- HuggingFace 会加价 30-50%
- 多一层路由，延迟更高
- 模型选择受限

## 成本对比

| 模型 | HuggingFace | DeepSeek 官方 | 节省 |
|------|-------------|---------------|------|
| Chat V3 | ~$0.20 | $0.14 | 30% |
| R1 | ~$0.80 | $0.55 | 31% |
| Coder | ~$0.20 | $0.14 | 30% |

## 功能特性

- ✅ 完整的 TypeScript 类型支持
- ✅ 49 个测试用例，CI 通过
- ✅ 实时成本追踪
- ✅ 流式响应
- ✅ 智能模型选择
- ✅ 批量处理

## 快速开始

```bash
npm install @openclaw/provider-deepseek
```

配置：
```json5
{
  env: { DEEPSEEK_API_KEY: "sk-..." },
  agents: {
    defaults: {
      model: { primary: "deepseek/deepseek-chat" }
    }
  }
}
```
