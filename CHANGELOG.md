# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-05

### Added
- Initial release of DeepSeek Provider for OpenClaw
- Support for all DeepSeek models:
  - deepseek-chat (V3)
  - deepseek-reasoner (R1)
  - deepseek-coder
- Direct API integration without middlemen
- OpenClaw adapter for seamless integration
- Full TypeScript support with type definitions
- Comprehensive test suite (25 tests)
- Cost tracking and comparison features
- Streaming response support
- Reasoning model support with chain-of-thought
- Complete documentation and examples

### Features
- **Cost-Effective**: 25-30% cheaper than HuggingFace route
- **Fast**: Direct API connection, no routing overhead
- **Complete**: Full access to all DeepSeek capabilities
- **Type-Safe**: Full TypeScript support
- **Well-Tested**: 100% test coverage on core functionality

### Documentation
- README with quick start guide
- OpenClaw integration documentation
- API reference and examples
- Troubleshooting guide
- Cost comparison tables
