# DeepSeek Provider - Basic Usage Example

This example demonstrates basic usage of the DeepSeek Provider.

## Setup

```bash
npm install
```

## Run

```bash
export DEEPSEEK_API_KEY="sk-..."
npm start
```

## What it demonstrates

1. Simple chat completion
2. Cost-optimized model selection
3. Cost estimation and tracking
4. Savings calculation vs HuggingFace
